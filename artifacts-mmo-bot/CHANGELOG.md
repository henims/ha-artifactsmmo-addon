# Changelog

## latest (Rolling Release)

This add-on uses rolling releases. Each push to main triggers a new build.
Check the add-on logs for build number to identify which version is running.

### Recent Changes
- Build info logged at startup (Build #, commit SHA, date)
- Automatic source sync from main repository
- Bug fixes and improvements

## 1.0.0 (2024-12-06)

- Initial release
- Automated bot for ArtifactsMMO game
- Web dashboard with real-time status
- Ingress support for Nabu Casa remote access
- Multi-character support
