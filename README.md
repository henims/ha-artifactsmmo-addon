# ArtifactsMMO Home Assistant Add-on

Automated bot for the ArtifactsMMO game with a web dashboard.

## Installation

1. Add this repository to Home Assistant:
   - Go to **Settings** → **Add-ons** → **Add-on Store**
   - Click the three dots (⋮) in the top right → **Repositories**
   - Add: `https://github.com/henims/ha-artifactsmmo-addon`

2. Find "ArtifactsMMO Bot" in the add-on store and install it

3. Configure your API token in the add-on settings

4. Start the add-on

5. Access the dashboard via the sidebar or through Nabu Casa

## Configuration

| Option | Description |
|--------|-------------|
| `api_token` | Your ArtifactsMMO API token |

## Features

- Automated character management
- Web dashboard with real-time updates
- Ingress support (works with Nabu Casa)
- Multi-character support
